tags e suas funcionalidades

  • <h1> a <h6>: tags de título, do maior ao menor.
  • <p> </p>: tag de parágrafo, usado para criar um parágrafo de texto.
  • <ol> </ol>: lista ordenada, usando números para listar.
  • <ul> </ul>: lista não ordenada, apenas pontinho.
  • <style> </style>: representa o css, tudo dentro desta tag é designe.
  • <br>: tag de quebra de linha.
  • <hr>: cria linha horizontal na tela.
  • <strong>: tag para negrito.
  • <em>: tag para itálico
  • <mark> </mark>: tag para marcar texto.
    Importante destacar que naturalemente a cor será amarelo, porém é possível reocar usando o atributo style="background-color: blue"
  • <del></del>: tag para riscar em baixo.
  • <ins></ins>: tag de grifar.
  • <abbr></abbr>: abreviação, exemplo BR
