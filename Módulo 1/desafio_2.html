DESENVOLVIMENTO WEB

>HTML

Linguagem de marcação que cria o conteúdo do site.

Tags

Elementos utilizados para estruturar e organizar o site.

Como usar tag?

As tags são são representadas por < e >

>CSS

Linguagem de marcação que define o estilo do site.

Tipos de Style

Inline, interno e externo

Inline

Estilo aplicado na mesma linha da tag html que se deseja estilizar (menos recomendável)

Interno

Aplicado dentro do mesmo documento HTML, porém, na região superior do documento

Externo

Aplicado em documento css próprio e criado link (mais recomendado).
